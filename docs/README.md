# Documentation Home

This documentation is split by audience and operational goal.

| Document | Audience | Purpose | Time |
|---|---|---|---|
| [User Guide](user-guide/README.md) | End users / operators | Install, run, and interpret outputs | ~15 min |
| [Developer Guide](developer-guide/README.md) | Contributors | System internals and extension workflow | ~25 min |
| [Architecture](architecture/README.md) | Developers / architects | Runtime topology and component interactions | ~20 min |
| [Research Loop](research-loop/README.md) | Developers / operators | Loop lifecycle, states, and control logic | ~15 min |

## Recommended Reading Paths

- **New user**: User Guide → Architecture (overview section only).
- **New contributor**: Developer Guide → Architecture → Research Loop.
- **On-call/debugging**: Research Loop → Developer Runbook.
