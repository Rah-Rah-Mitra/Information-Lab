# Foundations of Linear Algebra for Iterative Methods

*This note covers the fundamental concepts of linear algebra required for understanding iterative solvers, including matrix structures, eigenvalues, and projection operators. It provides the mathematical basis for analyzing convergence and stability in large-scale scientific computing. The concepts are essential for transitioning from direct to iterative solution methods.*

This note provides a foundational overview of the mathematical principles of [[Linear Algebra]] necessary for the study of iterative methods for large sparse linear systems.

## Concept
[[Linear Algebra]] provides the framework for understanding matrix operations, spectral properties, and subspace projections. Key concepts include:

### Eigenvalues and Eigenvectors
An [[Eigenvalue]] λ and a corresponding [[Eigenvector]] $v$ satisfy the relation:
$$ Av = λ v $$
The set of all eigenvalues is called the spectrum of $A$.

### Matrix Norms and Inner Products
An [[Inner Product]] $g(x, y)$ on a complex vector space is a mapping that is Hermitian, positive definite, and linear with respect to the first argument. A [[Matrix Norm]] is typically induced by vector norms, such as the [[Euclidean Norm]].

### Matrix Structures
Certain classes of matrices have specific properties that influence the efficiency of iterative solvers:
- [[Normal Matrix]]: Satisfies $A^*A = AA^*$.
- [[Hermitian Matrix]]: A [[Normal Matrix]] whose eigenvalues are real.
- [[Positive Definite Matrix]]: A real matrix where $x^T A x > 0$ for all nonzero $x$.
- [[Sparse Matrix]]: A matrix with many zero elements, often represented via graph structures.

### Projection Operators
nA [[Projection Operator]] is a linear mapping that is idempotent, i.e., $P^2 = P$. An [[Orthogonal Projector]] is a projector that is also Hermitian ($P = P^*$).

### Decomposition and Subspaces
- [[QR Decomposition]]: A method to factorize a matrix into an orthogonal matrix $Q$ and an upper triangular matrix $R$.
- [[Krylov Subspace]]: A subspace generated by powers of a matrix applied to a vector, central to modern iterative methods.
